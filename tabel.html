Features of VS code

1.Code completion: IntelliSense is a set of features that includes code completion, parameter info, quick info, and member lists. This helps reduce the amount of code a developer needs to write manually.
2.Syntax highlighting: VS Code supports syntax highlighting for hundreds of languages.
3.Debugging: VS Code supports debugging.
4.Version control: VS Code has embedded version control with Git.
5.Extensions: VS Code has a variety of extensions that allow you to customize your experience. For example, the GitHub Copilot extension uses AI to help you write code.
6.Flexibility: You can modify your theme, keypads, settings, and create snippets.
7.Other features: VS Code also supports snippets, code refactoring, bracket-matching, auto-indentation, and box-selection.
8.Command Palette: You can access all commands by typing keywords in the Command Palette.

Shoutcuts Keys

shoutcuts Description
Ctrl + S Save the current file
Ctrl + O Opens the selected file
Ctrl + C copy the selected text
Ctrl + V paste the selected text
Ctrl + X cut the text
Ctrl + Y Redo the previously undone action
Ctrl + Z Cut the selected text or item
Ctrl + A Select all content in the current file
Ctrl + Shift + P Open Command Palette
Ctrl + Shift + N New window
Ctrl + Shift + W Close window
Ctrl + Shift + Alt + Down Add cursors to multiple lines for simultaneous editing
Alt + Up Arrow Move the selected line or block of code one line up.
Alt + down Arrow Move the selected line or block of code one line down.
